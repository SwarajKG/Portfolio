# 3D Interactive Developer Portfolio - Swaraj Gupta

A modern, high-performance, and fully responsive 3D portfolio website built for **Swaraj Gupta**, Senior Full Stack .NET Developer. This project features a professional dark theme, smooth animations, and interactive 3D elements.

## 🚀 Live Preview Features

- **Interactive 3D Hero:** A dynamic particle system built with **Three.js** that reacts to mouse movement.
- **Glassmorphism UI:** Modern frosted-glass aesthetic for panels and cards.
- **Scroll Animations:** Powered by **GSAP (GreenSock)** and **ScrollTrigger** for sophisticated reveal effects.
- **3D Tilt Effects:** Interactive hover states on experience and education cards using **Vanilla Tilt**.
- **Custom Cursor:** A sleek, animated cursor follower for enhanced user interaction.
- **Fully Responsive:** Optimized for Desktop, Tablet, and Mobile devices with a custom mobile navigation menu.
- **Performance Optimized:** Uses CDN-hosted libraries and optimized CSS for fast loading.

## 🛠️ Built With

- **HTML5:** Semantic structure and SEO-friendly markup.
- **CSS3:** Custom variables, Flexbox/Grid, and advanced animations.
- **JavaScript (ES6+):** Core logic and interactivity.
- **Three.js:** 3D particle background rendering.
- **GSAP:** Industry-standard animation engine.
- **FontAwesome:** Professional developer icons.
- **Google Fonts:** Inter and Fira Code for a clean "developer" look.

## 📂 Project Structure

This project follows a flat directory structure for simplicity and ease of deployment:

```text
Portfolio/
├── index.html    # Main entry point and structure
├── styles.css    # Modern dark theme and layout styles
├── script.js     # Interactivity, 3D logic, and animations
└── README.md     # Project documentation
```

## 💻 Getting Started

1. **Clone/Download** this repository to your local machine.
2. Ensure you have an active internet connection (to load the library CDNs).
3. Open `index.html` in any modern web browser (Chrome, Firefox, Edge, Safari).

## 🔧 Customization

- **Changing Colors:** Update the CSS variables in the `:root` section of `styles.css`.
- **Updating Content:** Modify the text within the semantic sections of `index.html`.
- **Adjusting 3D Effects:** Fine-tune the particle count or rotation speed in the `Three.js` section of `script.js`.
